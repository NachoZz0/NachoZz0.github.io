如何在Android上运行PPSR

首次安装

1。下载并安装 hsr apk(https://gofile.io/d/wbtgyY (Password: Excalibur)下载CN)
2。安装 termux(登录后下载)
3。复制以下命令 

pkg install wget -y && pkg install zip -y && wget -O ppSR_Android.zip https://tinyurl.com/46cdha9h && unzip -o ppSR_Android.zip && chmod +x install.sh && ./install.sh

4。打开Termux并允许通知
4。粘贴先前复制的命令，然后按Enter(须打开你的VPN这里提供一个:下载)
5。允许所有存储权限的提示，向下滚动到Termux并选择它，然后允许所有存储权限（这是您可以稍后同步）
6。等待屏幕上的内容完成执行

如何运行？

1。输入./run.sh并按Enter(注意不要少了.)
2。打开 hsr beta
3.选择密码登录,随便填写，等待资源下载完成，enjoy
