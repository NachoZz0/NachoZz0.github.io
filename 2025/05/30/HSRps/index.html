HSR private server for Android

2025/5/30 hsr 游戏

HSR3.4beta安卓本地私服

下载即用，小白也能马上上手

1.准备：需约15G空间下载HSR3.4.x的资源
2.下载HSR3.4.2(3.4v2)客户端HERE(Password: Excalibur)和私服服务端HERE(登录后下载)fromgatoSR
3.安装好后打开服务端(spaghetti什么什么的那个)点击start server
4.打开游戏，点密码登录，随便填写，下载好资源后即可享受

功能

仅仅是一个简单的ps，功能完整的私服应该不会再公开了(上版本才一个leak大手子才被mhy制裁了)
地图，战斗（可在srtools(需开VPN)编辑角色配置）（→如何配置角色）
忘却，虚构等可以进入

问题

许多功能并没有修复，点了会一直转圈，退出游戏重进即可(别乱点)
1001_1登录异常，关闭服务端和游戏后重进
全局分发错误，没开服务端

如何配置角色

打开srtools(需开VPN)
点击connectps→connect
修改你想要的配置后点sync即可
默认配置为5星0+1，4星6+5
全6+5下载
下载JSON后点srtools左下角的设置按钮→Add database|upload freesr****.json 上传刚才下载的JSON，点upload freesr****.json上面的东西切换刚才上传的配置
返回，点robinsr→sync，回到游戏后配置应该已经生效了
