GFW

2025/6/21

长城防火墙（简称 GFW）是中国政府用来控制和过滤互联网内容的国家级网络系统。下面我用常人都能理解的方式来解释它的作用和原理——

🌐 GFW 的作用

  1. 屏蔽境外“敏感”信息
    比如访问 Google、Facebook、YouTube、Twitter、Wikipedia 等国外网站，往往会直接打不开 (techtarget.com, blog.luckysix.cc, en.wikipedia.org)。

  2. 过滤敏感内容
    系统会检测关键词或链接里出现敏感词，一旦发现就阻断连接 。

  3. 遏制 VPN、代理穿透
    不让用户轻易绕过它设下的防线 (experte.com)。

简言之，它就是为了“净化可访问的互联网内容”，防止用户接触某些信息，同时也推动国内替代平台的发展 (en.wikipedia.org)。

🛠 GFW 的工作原理

1. DNS 污染

当你输入 google.com 时，GFW 会偷偷修改 “查地址”（DNS）的结果，给你一个不存在或错误的 IP，让你根本找不到网站 (jinhui1988.github.io)。

2. IP 路由封锁

目标网站的 IP 地址被设置成“死路”或“黑洞”，从根本上断掉访问 (zh.wikipedia.org)。

3. 关键词过滤 + TCP 重置

如果你访问的网址或网页里出现敏感词，GFW 会“监听”你在发送什么内容，然后马上发一个假的“断线重置（RST）”指令，让你的连接瞬间断掉 (zh.wikipedia.org)。

4. 深度数据包检测（DPI）

GFW 不仅能看明文，还能通过技术手段识别出 VPN、加密协议甚至 Tor 的特征，并加以封堵 (jinhui1988.github.io)。

5. 持续“猫抓老鼠”

GFW 不断升级其技术，VPN 服务和 Tor 也不断改进。它们之间是一场持续的“攻防竞赛” 。

🔁 生动比喻

想象你要从国内去国外网站：

  • DNS污染：系统告诉你“那个国外商店不存在”。
  • IP封锁：即使知道地址，你的车也被挡在了一条“死胡同”里。
  • 关键词过滤 + TCP重置：你开车赶到了商店门口，话还没说完，保安就突然关门，把你赶了出来。
  • 深度检测：甚至你换了车（VPN），保安也能认出来，把你拦住的不让进。

✅ 总结

长城防火墙就是一个多层级的网络“过滤器”，涵盖：

技术手段 作用说明
DNS污染 让你找不到网站
IP封锁 阻止直接访问
关键词过滤 + RST 检测敏感内容后断线
深度流量检测 识别并封锁 VPN/Tor

它的目标是控制网络内容流入，维护信息环境的“可控性”。这是一个高度自动化的系统，实时监控、实时封禁，非常复杂，任何想要“翻墙”的人，都相当于是挑战它的技术防线。
